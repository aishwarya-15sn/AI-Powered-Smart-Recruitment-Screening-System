# AI-Powered Smart Recruitment Screening System

## Overview
AI-Powered Smart Recruitment Screening System is an intelligent hiring assistant that automates resume screening using NLP and machine learning techniques.

The system analyzes resumes, extracts skills, compares them with job descriptions, and generates candidate matching scores to help recruiters shortlist candidates efficiently.

---

## Problem Statement
Manual resume screening is time-consuming and inconsistent. Recruiters often spend hours reviewing resumes that may not match job requirements.

This project aims to:

- Automate resume analysis
- Extract skills using NLP
- Match candidates with job descriptions
- Rank applicants based on relevance score
